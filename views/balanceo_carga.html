CDN y distribución de contenido

El CDN entrega imágenes, CSS y JS desde servidores cercanos al usuario, reduciendo latencia y tiempo de carga.

Balanceador de carga (LB)

Distribuye las peticiones entre dos o más servidores que contienen el mismo front-end. Esto evita que un solo servidor se sature y garantiza continuidad en caso de falla.

Servidores redundantes

  • Servidor A: Atiende clientes normalmente.
  • Servidor B: Respaldo en caso de falla de A.

Si uno falla, el otro atiende al usuario sin interrumpir el servicio.

Beneficio para campañas de alta demanda

Durante promociones especiales (Día de la Tierra, Buen Fin), el tráfico puede duplicarse. Balanceo + CDN asegura que el catálogo de panes y cafés no se caiga y los usuarios tengan buena experiencia.
